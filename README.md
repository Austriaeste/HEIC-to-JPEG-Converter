# HEIC to JPEG Converter

ブラウザ上で完結する、軽量かつセキュアなHEIC画像変換ツールです。

## 概要

本ツールは、HEIC形式（iPhoneなどで標準の画像形式）をJPEG形式に変換するクライアントサイド専用のツールです。WebAssembly (Wasm) を活用し、**すべての処理をブラウザ内で行うため、インターネット環境がなくても動作し、画像データが外部サーバーへ送信されることもありません。**

## 特徴

* **オフライン動作**: ネット接続が不要な環境でも問題なく動作します。
* **高いセキュリティ**: 画像データはローカル（PCのブラウザ内）から一歩も外に出ません。社外秘の画像も安心して変換可能です。
* **環境を選ばない**: インストール不要。Chrome, Edge などのモダンブラウザで開くだけですぐに使えます。
* **シンプルな操作**: ドラッグ＆ドロップで素早く変換できます。

## 使い方

1. 本ファイルをダウンロードし、ブラウザで開きます。
2. 変換したい `.heic` ファイルを画面のドロップエリアにドラッグ＆ドロップします。
3. 自動的にJPEG変換が開始され、ダウンロードフォルダに保存されます。

## 技術スタック

* **WebAssembly (Wasm)**: 高速な画像デコード処理に使用
* **heic2any**: HEIC形式の変換ライブラリ
* **Vanilla JS / CSS**: シンプルで軽量なUI実装

## 注意事項

* 一度に大量のファイルをドラッグ＆ドロップすると、ブラウザのセキュリティ制限によりダウンロードがブロックされる場合があります。その際は、数回に分けて実行してください。
* 本ツールはブラウザのメモリを使用します。非常に高解像度な画像や、膨大な枚数の同時変換はメモリ不足を引き起こす可能性があるためご注意ください。
